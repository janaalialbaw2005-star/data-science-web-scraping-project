# Data Science 

## Project Title
Web Scraping and Data Analysis using Python

## Project Description
This project demonstrates data collection from a real-world website using web scraping techniques. The collected data is stored in JSON format and analyzed using Pandas. Multithreading is applied to improve data collection performance.

## Tools and Technologies
- Python
- Requests
- BeautifulSoup
- Pandas
- JSON
- Multithreading (ThreadPoolExecutor)

## Project Workflow
1. Scrape data from a public website using Requests and BeautifulSoup.
2. Apply multithreading to speed up data collection.
3. Save the collected data in JSON format.
4. Analyze the data using Pandas.
